## Описание
В данной лабораторной работе реализован алгоритм сортировки элементов по убыванию их длины. 
Алгоритм сортировки: Пузырьковая сортировка (Bubble Sort). Скрипт принимает строку слов, разбивает её на список и упорядочивает элементы от самых длинных к самым коротким путём попарного сравнения и перестановки соседних элементов.
