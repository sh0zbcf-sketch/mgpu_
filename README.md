## Описание
В данной лабораторной работе реализован базовый алгоритм горизонтального масштабирования данных — шардинг на основе расчета остатка от деления.
Алгоритм шардинга: Программа принимает уникальный идентификатор пользователя (user_id) и общее количество доступных серверов (servers_count). На основе формулы user_id % servers_count определяется конкретный целевой сервер (шард) для распределения и хранения данных.
